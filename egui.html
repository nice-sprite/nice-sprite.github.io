EGUI Renderer

Before I started working on LUI Editor, I was investigating the Rust ecosystem and found bgfx and egui as candidates for the technology components I would use in the final product. Below is a the demo program I wrote to test what it would be like to use egui. Everything is rendered using the Rust bindings for bgfx.

In the end, I decided that the Rust ecosystem was not mature enough to support what I wanted to accomplish and retired the idea. I did, however, enjoy combining these two libraries and learned a lot while doing it!
